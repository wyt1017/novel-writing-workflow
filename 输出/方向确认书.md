# W04 方向确认书

## 篇幅选择

| 选项 | 字数范围 | 章节数 | 适合场景 | 选择 |
|------|----------|--------|----------|------|
| 短篇 | 10-30万字 | 50-150章 | 练手/试水 | |
| 中篇 | 30-60万字 | 150-300章 | 单一故事线 | |
| **中长篇** | **100-200万字** | **500-1000章** | **网文主流，收益最大化** | ✅ |
| 超长篇 | 200-500万字 | 1000-2500章 | 顶级作者/IP向 | |

**选择理由：** 系统流日常向适合长线运营，章节数够多才能充分展开"隐藏身份的层层揭露"和"配角的持续脑补"。

## 故事核心设定

| 维度 | 内容 |
|------|------|
| 主角身份 | 22岁普通社畜，二本毕业，在一家普通公司做普通文员 |
| 金手指 | 「每日签到系统」— 每天签到获得随机能力/物品/属性，但系统发布的任务强制要求"保持低调" |
| 核心矛盾 | 能力不断变强 vs 被系统强制低调 → 被迫在日常中隐藏实力 |
| 主线驱动 | 签到天数累积 → 解锁更高级签到奖励 → 逐渐揭开系统来源的秘密 |
| 冲突来源 | ① 日常生活中的装逼打脸需求 vs 系统惩罚机制；② 配角对主角的持续脑补 vs 真相；③ 隐藏势力的逐渐浮现 |

## 一句话概括

> "签到十年，我拥有了毁灭世界的能力——但系统说我必须继续当社畜。"

## 故事核（扩展三段式）

| 阶段 | 核心内容 |
|------|----------|
| **开局（1-50章）** | 主角获得系统→不适应低调要求→日常生活中被动装逼→配角开始脑补→第一个身份被误会 |
| **展开（51-200章）** | 系统任务升级→被迫卷入更大事件→隐藏身份被逐步揭露→势力开始注意到他→脑补升级为"都市传说" |
| **结局（201-500章）** | 系统来源揭晓→最终势力对决→身份完全暴露→选择继续低调还是接受真实身份 |

## 可扩展情节方向（≥3个）
1. 同事/邻居/前女友各自的脑补支线 → 形成"多视角叙事"的喜剧效果
2. 签到系统每隔100天解锁"隐藏身份碎片"→ 拼图式揭露真相
3. 系统偶尔给出"高调任务"（惩罚性的）→ 主角被迫短暂展现实力再装回去
4. 出现另一个系统持有者 → 竞争关系
5. 系统管理员（AI）逐渐产生自我意识 → 与主角形成羁绊
