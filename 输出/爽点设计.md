# W05 爽点设计

## 爽点分类选择

| 类别 | 说明 | 选择 | 理由 |
|------|------|------|------|
| ① 实力碾压 | 主角隐藏的实力碾压对手 | ✅ | 核心爽点，签到十年积累的能力突然展露 |
| ② 身份揭露 | 隐藏大佬身份被揭露 | ✅ | 层层揭露，从"普通社畜"到"都市传说" |
| ③ 打脸逆袭 | 被轻视后强势反击 | ✅ | 日常场景中的被动打脸（非主动装逼） |
| ④ 智谋布局 | 智商碾压、运筹帷幄 | — | 不适合本作风格 |
| ⑤ 情感征服 | 攻略、情感线 | ✅ | 脑补型感情线（配角单方面倾慕） |
| ⑥ 资源获取 | 签到获得稀有物品/能力 | ✅ | 驱动日常剧情的基础爽点 |

**选择：①②③⑤⑥（5种）**

## 爽点模式设计

### 模式A：签到获得 → 日常隐忍 → 被迫展露 → 配角脑补 → 身份升级

| 阶段 | 时长控制 | 具体操作 |
|------|----------|----------|
| **获得期** | 开头500字 | 今日签到内容：随机能力/物品（有随机性刺激） |
| **铺垫期** | 2-3章 | 新能力让主角更方便，但他刻意不用→压抑感 |
| **触发期** | 1-2章 | 外部事件（同事被欺负/偶遇事件）→ 被迫出手 |
| **爆发期** | 1章内 | 能力展露（但主角努力伪装成"运气好/巧合"） |
| **余韵期** | 1-3章 | 配角们脑补真相→信息差喜剧→主角继续装没事 |

### 模式B：身份揭露弧（每50章一个大节点）

| 阶段 | 章节点 | 揭露内容 |
|------|--------|----------|
| 第1层身份 | ~50章 | "他好像不是普通人"→ 同事圈内小范围关注 |
| 第2层身份 | ~100章 | "他是不是隐藏的大佬"→ 公司高层注意 |
| 第3层身份 | ~150章 | "他的过去有秘密"→ 势力开始调查 |
| 第4层身份 | ~200章 | "他可能是那个传说中的……"→ 都市传说级 |

## 爽点密度规划

| 阶段 | 章节范围 | 密度 | 说明 |
|------|----------|------|------|
| 开局 | 1-30章 | 每3章1个小爽点 + 每10章1个大爽点 | 高频留住读者 |
| 展开 | 31-100章 | 每5章1个小爽点 + 每15章1个大爽点 | 适中稳定 |
| 中期 | 101-300章 | 每7章1个小爽点 + 每20章1个大爽点 | 配合剧情节奏 |
| 后期 | 301-500章 | 每5章1个小爽点 + 每10章1个大爽点 | 临近结局加速 |

## 爽点独特设计（防同质化）

| 常规套路 | 本作反套路 |
|----------|------------|
| 签到→变强→主动装逼 | 签到→变强→被迫低调→被动流露→他人脑补 |
| 系统发布"击杀/征服"任务 | 系统发布"按时上班/参加团建/给妈妈打电话"任务 |
| 实力碾压后炫耀 | 实力碾压后尴尬解释"这是运气/巧合/碰巧" |
| 女角色主动倒贴 | 女角色单方面脑补→主角毫不知情→信息差喜剧 |
