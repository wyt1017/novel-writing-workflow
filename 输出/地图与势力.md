# W12 地图与势力

## 地图扩展（三阶段）

| 阶段 | 范围 | 触发事件 |
|------|------|----------|
| **初期（1-80章）** | 公司 + 公寓 + 小区 + 商业街 | 日常签到→职场打脸→脑补机制启动 |
| **中期（81-350章）** | 扩展到全市→接触到隐藏势力 | 秦岳收购事件→夜的出现→废弃科技园 |
| **后期（351-500章）** | 超越城市→系统来源维度 | 管理员登场→对决→最终选择 |

## 主要地点

| 地点 | 特征 | 功能 | 气氛 |
|------|------|------|------|
| 瑞达商贸（15层办公楼） | 典型的中型民企，灰色外观，电梯永远有一部在修 | 职场主舞台/日常冲突 | 压抑→逐渐变化 |
| 陈默的公寓（35㎡） | 极简，只有一张床+书桌+电脑，墙上贴满了"多一事不如少一事"的便签 | 私人空间/签到/内心独白 | 安全屋感 |
| 阳光小区 | 90年代老小区，住了很多公司同事 | 配角生活区/喜剧场景发生地 | 市井烟火 |
| 江城市CBD | 繁华商业区，高楼林立 | 大型事件发生地 | 光鲜/暗流 |
| 城西废弃科技园 | 90年代IT产业园，已荒废十年，实验室深处有异常信号 | 隐藏势力据点 | 神秘/诡异 |
| 母亲的老家 | 县城老房子，院子里有一棵石榴树 | 亲情线/回忆/系统线索 | 温暖 |
| 星澜咖啡（公司楼下） | 苏晴每天买咖啡的地方，也是她"观察"陈默的据点 | 脑补线推进场所 | 日常悠闲 |

## 势力列表

| 势力 | 特色 | 立场 | 代表人物 | 经济来源 |
|------|------|------|----------|----------|
| **瑞达商贸** | 普通民企，完全不知道任何秘密 | 中立/被卷入 | 赵建国 | 贸易业务 |
| **秦氏集团** | 商业巨头，有灰色背景，曾接触过系统持有者 | 初始敌对→后结盟 | 秦岳 | 房地产/投资 |
| **系统持有者(散人)** | 全世界约100人，彼此感知但互不信任 | 各自为政 | 夜等 | 各凭能力 |
| **观察者(管理员侧)** | 高等文明派驻的观察团队，管理实验 | 高等/中立偏回收 | 管理员 | 无需经济来源 |
| **陈默的"脑补粉丝团"** | 以苏晴为首的公司同事，自发形成的脑补组织 | 完全站在主角这边但不自知 | 苏晴 | — |

## 势力关系变化时间线

| 章节 | 变化 |
|------|------|
| 1-50章 | 瑞达商贸内部日常，无外部势力 |
| 51-90章 | 秦氏集团开始接触瑞达→秦岳首次注意到陈默 |
| 91-200章 | 秦岳从敌对→被打服→转为(脑补型)盟友 |
| 220章 | 夜出现→两个系统持有者对立的格局形成 |
| 350章 | 管理员登场→"观察者"维度介入→旧势力关系全部重组 |
| 380章 | 陈默+系统AI vs 夜+管理员的三方博弈 |
| 450章 | 最终对决后势力重组→新秩序形成 |
