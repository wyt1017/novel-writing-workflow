# W10 世界观设定

## 时代背景

| 维度 | 内容 |
|------|------|
| 时间 | 2024年，当代都市 |
| 空间 | 中国某新一线城市（江城市，虚构） |
| 对故事的影响 | 现代都市让"隐藏大佬"的设定更可信——在繁华都市里，一个看起来普通的人可能真的藏着惊天秘密 |
| 世界类型 | 表面是普通现代都市，暗面存在"系统持有者"的隐秘世界（但占比极低，全城只有2-3人） |

## 地理格局

| 地点 | 功能 | 出场章节 |
|------|------|----------|
| 瑞达商贸公司（15层办公楼） | 主角职场主舞台 | 1-500章（贯穿） |
| 陈默租住的小公寓（一居室，35㎡） | 主角私人空间、系统签到场所 | 1-500章（贯穿） |
| 阳光小区（老家属区） | 配角群的居住地 | 5-200章 |

| 母亲的老家（县城） | 亲情线/系统线索 | 电话贯穿，实地50/150章 |
| 废弃科技园区（城西） | 隐藏势力的秘密基地 | 350-450章 |

## 社会结构

| 层面 | 内容 |
|------|------|
| 普通社会 | 正常都市生活——公司/家庭/社交，99.99%的人不知道系统的存在 |
| 隐秘层面 | 全世界只有极少数人拥有系统（本作设定：约100人），彼此之间通过系统可以互相感知但主动隐藏 |
| 权力体系 | 实际的权力来自系统持有者的个人实力，与明面上的社会地位不一定重合。管理员为代表的观察者在更高维度 |

## 核心规则

| 规则 | 内容 |
|------|------|
| 系统来源 | 高等文明的"人类潜力观察实验"——系统是实验工具，通过向随机人类投放来收集数据 |
| 系统规则 | 每个系统独立运行，类型不同（签到型/任务型/竞争型等），互不兼容 |
| 力量暴露限制 | 主动暴露系统存在 → 扣除所有奖励；但被动被发现不在此限（这也是为什么主角可以"被迫展露"，只要他自己不承认） |
| 社会态度 | 普通人不知道系统存在，但会对"异常现象"产生认知失调——大脑自动合理化（所以陈默的各种"碰巧"会被解读为运气好而不是超能力） |
| 系统持有者关系 | 两个系统持有者距离过近时会互相感知到，但感知不精确（不会知道对方是谁，只知道"附近有人"） |

## 隐藏设定（4级悬念）

| 级别 | 内容 | 揭晓方式 | 揭晓章节 |
|------|------|----------|----------|
| 第1级 | 陈默不是普通人 | 配角们逐步发现 | 10-80章 |
| 第2级 | 世界上存在系统持有者 | 第二个持有者出现 | 220章 |
| 第3级 | 系统的真正来源 | 管理员登场 | 350章 |
| 第4级 | 签到系统的AI产生了自我意识 | 系统AI主动对话 | 380章 |
