# W11 力量体系

## 体系类型：都市系统流

核心力量来源：每日签到系统，通过累积签到天数解锁不同等级奖励。

## 签到等级划分

| 签到天数 | 等级名称 | 标志能力 | 战力对标 | 寿命 |
|----------|----------|----------|----------|------|
| 0-365天 | 新手期 | 基础属性微幅增长 | 略强于常人 | 正常 |
| 366-1095天 | 成长期 | 获得特殊技能（范围随机） | 特种兵级 | 正常 |
| 1096-2555天 | 成熟期 | 技能可自主选择方向 | 小型武器级 | 正常+20年 |
| 2556-3650天 | 巅峰期 | SSS级能力解锁 | 导弹级 | 正常+50年 |
| 3651-5475天 | 传说期 | 规则级能力 | 天灾级 | 正常+100年 |
| 5476天+ | 神级 | 创造/修改系统规则 | 概念级 | 半永生 |

## 主角当前状态（第1章）

| 参数 | 数值 |
|------|------|
| 签到天数 | 3650天（刚满十年） |
| 当前等级 | 巅峰期（第3650天是巅峰→传说的门槛日） |
| 已获得随机技能 | 约200+个（大部分是生活技能） |
| SSS技能 | 「绝对冷静」「极限计算」「语言通晓」——都是被动/辅助型 |
| 尚未解锁 | 攻击型SSS技能（将在第10章首次获得——而主角非常头疼这件事） |

## 突破条件

| 突破 | 要求 | 风险 | 成功率 |
|------|------|------|--------|
| 新手→成长 | 签到365天 | 无 | 100% |
| 成长→成熟 | 签到1095天 + 完成50个系统任务 | 失败任务累积 → 属性扣减 | 自动 |
| 成熟→巅峰 | 签到2555天 + 被动展露≥10次（系统追踪"被发现"次数） | 展露不足则不晋升 | 自动 |
| 巅峰→传说 | 签到3650天 + 系统AI觉醒（需外部触发） | AI觉醒条件未知 | 未知（第350-380章触发） |
| 传说→神级 | 获得系统管理员权限 | 需要击败或说服当前管理员 | 单人概率极低 |

## 战力对比表

| 对手类型 | 主角需要展露的实力 | 典型场景 |
|----------|-------------------|----------|
| 普通混混(5人) | 不使用能力即可（79点力量随便打） | 第1-10章 |
| 专业保镖(持械) | 使用1-2个被动技能 | 第50-80章 |
| 特种兵级(退役特种兵) | 略微主动使用技能 | 第150章 |
| 另一个系统持有者(同级) | 需要用到SSS技能 | 第220-300章 |
| 管理员(更高维) | 需要系统AI觉醒配合 | 第380-450章 |

## 系统任务的"惩罚机制"（独特设计）

| 任务类型 | 示例 | 惩罚（未完成） | 设计意图 |
|----------|------|---------------|----------|
| 日常任务 | "按时上班不迟到" | 扣除随机属性3点 | 强制主角保持"普通人"生活节奏 |
| 社交任务 | "参加公司团建不早退" | 扣除魅力值5点 | 增加喜剧场景 |
| 亲情任务 | "给妈妈打电话" | 扣除幸运值10点 | 埋亲情线 |
| 隐藏任务 | (偶尔出现)"帮助一个陌生人" | 冻结签到7天 | 打开支线剧情 |

## 特殊机制

| 机制 | 内容 |
|------|------|
| **被动展露统计** | 系统会自动记录主角"被他人察觉异常"的次数——这是晋升传说级的条件之一。主角每次想低调，但系统的晋升机制在反向推他"被注意" |
| **冷却机制** | SSS技能有24小时冷却，且使用后24小时内系统会发布一个"补偿性低调任务" |
| **系统AI成长** | 签到3650天后，系统AI开始从"机械执行"向"自主意识"转变——这是一个隐性伏笔 |
