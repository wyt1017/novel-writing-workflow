# W13 开篇设计

## 300字开篇（黄金三章第一章开头）

---

早晨七点半，闹钟响了第三遍。

陈默从被窝里伸出手，摸到手机，按掉。屏幕弹出通知——

> **「每日签到系统」**
> **签到天数：3650天**
> **今日签到：已完成**
> **今日奖励：SSS级技能「极限计算」——可在0.01秒内完成任意复杂度的数学运算**
> **今日任务：按时上班，不迟到**
> **任务奖励：随机属性+2**
> **失败惩罚：扣除智力值10点**

他看了一眼，划掉。

十年了。从小学六年级点到大学毕业，这个弹窗从来没有消失过。他试过换手机、刷系统、恢复出厂设置——没用。它就像长在屏幕底层的东西，怎么甩都甩不掉。

至于那个SSS级技能……他打了三个哈欠，脑子里自动计算出了从出租屋到公司最快路线的概率学模型——17条路线，当前时间最优路径是翻墙穿过阳光小区后门，成功率87.3%。

他选择了第18条：正常走到公交站等车。

「多一事不如少一事。」

## 黄金三章规划

### 第一章（2000-3000字）

| 要素 | 内容 |
|------|------|
| 开场钩子 | 签到第3650天获得SSS能力→反应用平淡（勾住读者：这人怎么回事？） |
| 认识主角 | 通过早晨日常建立：社畜身份/佛系性格/隐藏实力 |
| 环境建立 | 公司、公寓、通勤路线 |
| 小冲突 | 上班路上偶遇意外事件→被迫使用能力但伪装成巧合 |
| 结尾钩子 | 他走后，监控室里一个人反复回放录像，表情逐渐凝重 |

### 第二章（2000-3000字）

| 要素 | 内容 |
|------|------|
| 深化矛盾 | 昨天的事件在公司里开始发酵→不同人给出不同解读 |
| 小爽点 | 部门主管王强为难他→能力悄无声息地反弹回去 |
| 脑补开始 | 苏晴注意到异常→脑补第一颗种子种下 |
| 结尾钩子 | 陈默收到系统弹出一行从未出现过的话：「检测到附近有其他持有者」 |

### 第三章（2000-3000字）

| 要素 | 内容 |
|------|------|
| 定调子 | 确立"日常+隐秘"的基调→轻松吐槽和暗流并存 |
| 大钩子 | 那个"其他持有者"的线索——是路过的，还是冲着他来的？ |
| 整体感觉 | 让读者确定：这是一个"被迫低调的大佬在都市里被脑补"的故事 |
