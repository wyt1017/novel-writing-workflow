## 节奏控制策略

| 阶段 | 节奏策略 | 章均冲突密度 |
|------|----------|--------------|
| 1-10章 | 快速建立+高频钩子（每章必须有钩子） | 极高 |
| 11-50章 | 日常+脑补交替，轻松为主 | 高→中 |
| 51-100章 | 外部势力逐步介入，紧张感上升 | 中→高 |
| 101-200章 | 商战+持有者暗线并行，张弛有度 | 高 |
| 201-300章 | 悬疑转强，节奏加快 | 高→很高 |
| 301-350章 | 决战前铺垫，逐步升温 | 很高 |
| 351-450章 | 巅峰对决，持续高强度 | 极高 |
| 451-500章 | 渐次回落，收束所有线索 | 高→低 |

## 避免节奏崩坏的措施

| 风险 | 措施 |
|------|------|
| 中段拖沓（100-200章） | 每15章安排1个大爽点+引入新角色/新悬念 |
| 日常过于平淡 | 日常总与脑补线/系统任务线交叠，每章都有至少1个"有趣的点" |
| 后期节奏过快 | 第5幕预留50章做收束，不仓促结尾 |
| 能力膨胀失控 | 能力解锁遵循签到天数限制，不可跳级 |
